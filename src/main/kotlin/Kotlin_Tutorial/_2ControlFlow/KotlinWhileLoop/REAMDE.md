
## Kotlin while Loop
Kotlin while loop is similar to Java while loop.

Syntax:
```kotlin
while(condition){
    //body of loop  
}  
```

Example:
```kotlin
fun main(args: Array<String>){  
    var i = 1  
    while (i<=5){  
        println(i)  
        i++  
    }  
}  
```

## Kotlin Infinite while Loop: vòng lặp vô hạn
```kotlin
fun main(args: Array<String>){  
        while (true){  
        println("infinite loop")  
        }  
}  
```
